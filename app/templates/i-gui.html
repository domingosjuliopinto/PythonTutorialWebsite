Pyhton GUI


A Graphical User Interface(GUI) is the first thing your user sees and interacts with when they opens your application or website. Having a good GUI goes a long way in increasing your platform's reputation and user count. A user interface usually includes a host of visual elements like icons, buttons, graphics, displayed text, and several other forms of input, like checkbox, text input boxes, and such.
It's the combination of all these elements that makes up a vital part of your application or website's user experience Input to these Visual elements can be from the usual mediums, such as keyboards, mice and touchscreens

TKinter

Python has a lot of GUI frameworks, but Tkinter is the only framework that's built into the
Python standard library. Tkinter has several strengths. It's cross-platform, so the same code
works on Windows, macOS, and Linux. Tkinter is lightweight and relatively painless to use compared to other frameworks.
The foundational element of a Tkinter GUT is the window Windows are the containers in which all other GI elements live These other it 1 elements, such as text boxes labels, and butions and laws as widgets Widgets are contained inside of windows.


Importing TKinter

With your Python shell open, the first thing you need to do is import the Python GUI Tkinter module:

>>> import tkinter as tk

A window is an instance of Tkinter's Tk class. Go ahead and create a new window and assign it to the variable window:

>>>window = tk.Tk()

When you execute the above code: a new window pops up on your screen.


Selection Widgets

Radiobutton: This widget displays a toggle button having an ON/OFF state. There may be more than one button, but only one of them will be ON at a given time.
Checkbutton: This is also a toggle button. A rectangular check box appears before its caption. Its ON state is displayed by the tick mark in the box which disappears when it is clicked to OFF
Combobox: This class is defined in the tik module of thioterpackage. It populates drop down data from a collection data type, such is a tuple or a list as vales parameter.
ListBox: Unlike ComboBox, this widget displays the entire collection of string items.The User can select one of multiple items.


Example: Button

# import everything from tkinter module
from tkinter import *
# create a tkinter window
root = Tk()
# Open window having dimension 100x100
root.geometry('100x100')
# Create a Button
btn = Button(root, text = 'Click me !', bd = '5', command = root.destroy)
# Set the position of button on the top of window.
btn.pack(side = 'top')
root.mainloop()
