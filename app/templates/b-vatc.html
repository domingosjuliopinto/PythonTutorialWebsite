Variables & Type Casting


Variables

Python has no command for declaring a variable.
A variable is created the moment you first assign a value to it.
Variables do not need to be declared with any particular type,
and can even change type after they have been set.
#check eg 1.

Variable names are case-sensitive.
i.e variable 'A' wont overwrite variable 'a'
#check eg 2.

String variables can be declared either by using single or double quotes
i.e x = 'john' is same as x = "john"




Type Casting

There may be times when you want to specify a type on to a variable.
This can be done with casting. Python is an object-orientated language,
and as such it uses classes to define data types, including its primitive types.
Casting in python is therefore done using constructor functions:
int() -
an integer number from an integer literal, a float literal (by removing all decimals),
or a string literal (providing the string represents a whole number)
float() -
constructs a float number from an integer literal, a float literal
or a string literal (providing the string represents a float or an integer)
str() -
constructs a string from a wide variety of data types,
including strings, integer literals and float literals

You can get the data type of a variable with the type() function
eg. print(type(x))
